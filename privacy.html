Политика конфиденциальности

Последнее обновление: 18 июня 2026 года

1. Общие положения

Настоящая Политика конфиденциальности описывает порядок обработки информации при использовании сервиса создания резюме онлайн.

2. Какие данные обрабатываются

Пользователь может самостоятельно вводить данные, необходимые для создания резюме, включая имя, контактную информацию, сведения об образовании, опыте работы, навыках и другую информацию.

3. Хранение информации

Для удобства пользователя данные могут временно сохраняться в локальном хранилище браузера (Local Storage).

Информация используется исключительно для работы сервиса и не передаётся третьим лицам.

4. Файлы Cookie

Сайт может использовать файлы cookie и аналогичные технологии для обеспечения корректной работы сервиса и анализа посещаемости.

5. Рекламные сервисы

В будущем сайт может использовать рекламные сервисы, включая Google AdSense. Такие сервисы могут применять cookie-файлы для отображения релевантной рекламы.

6. Изменения политики

Настоящая политика может периодически обновляться. Актуальная версия всегда публикуется на данной странице.

7. Контакты

По вопросам, связанным с работой сервиса или настоящей Политикой конфиденциальности, вы можете связаться с администрацией сайта через страницу контактов.
