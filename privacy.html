Политика конфиденциальности

Последнее обновление: 3 июля 2026 года

1. Общие положения

Настоящая Политика конфиденциальности описывает, какие данные собирает и обрабатывает сервис Resume Pilot при использовании сайта https://resumepilotonline.com.

Используя сайт, создавая аккаунт или оформляя доступ к PRO-функциям, пользователь соглашается с условиями данной Политики.

2. Какие данные мы собираем

При использовании сервиса мы можем обрабатывать следующие данные:

  • имя пользователя, если оно указано при регистрации;
  • адрес электронной почты;
  • зашифрованный пароль аккаунта;
  • статус аккаунта: FREE или PRO;
  • данные, которые пользователь вводит для создания резюме;
  • загруженные изображения, например фотографию для резюме;
  • технические данные: IP-адрес, браузер, устройство, язык, дата и время посещения;
  • данные аналитики и cookie-файлы.

Пароли пользователей не хранятся в открытом виде. Для защиты паролей используется хеширование.

3. Для чего используются данные

Данные используются для следующих целей:

  • создание и вход в аккаунт пользователя;
  • предоставление доступа к функциям сервиса;
  • активация и проверка PRO-статуса;
  • создание и экспорт резюме в PDF;
  • улучшение работы сайта и пользовательского опыта;
  • обеспечение безопасности сервиса;
  • обработка платежей и подтверждение оплаты.

4. Хранение данных

Данные аккаунта пользователя могут храниться в базе данных сервиса. Часть данных резюме также может сохраняться локально в браузере пользователя с помощью Local Storage для удобства работы с сайтом.

Данные хранятся только в объёме, необходимом для работы сервиса, предоставления функций аккаунта и выполнения технических задач.

5. Платежи

Оплата PRO-доступа обрабатывается через платёжного провайдера Stripe. Resume Pilot не хранит и не обрабатывает данные банковских карт пользователей на своих серверах.

Stripe может обрабатывать платёжные данные, имя, email, страну, IP-адрес и другую информацию, необходимую для проведения платежа, предотвращения мошенничества и выполнения юридических требований.

После успешной оплаты Resume Pilot получает только техническое подтверждение платежа и использует его для активации PRO-статуса аккаунта.

6. Cookie и аналитика

Сайт может использовать cookie-файлы и аналогичные технологии для корректной работы сервиса, сохранения пользовательских настроек, анализа посещаемости и улучшения сайта.

Resume Pilot может использовать Google Analytics и другие аналитические инструменты для понимания того, как пользователи взаимодействуют с сайтом.

7. Рекламные сервисы

Сайт может использовать рекламные сервисы, включая Google AdSense. Такие сервисы могут использовать cookie-файлы и похожие технологии для показа рекламы, оценки её эффективности и предотвращения злоупотреблений.

8. Сторонние сервисы

Для работы Resume Pilot могут использоваться сторонние сервисы:

  • Stripe — обработка платежей;
  • MongoDB Atlas — хранение данных;
  • Render — размещение backend-сервера;
  • Vercel — размещение frontend-части сайта;
  • Google Analytics — аналитика посещаемости;
  • Google AdSense — рекламные инструменты.

Эти сервисы могут обрабатывать данные в соответствии со своими собственными политиками конфиденциальности.

9. Передача данных третьим лицам

Resume Pilot не продаёт персональные данные пользователей третьим лицам.

Данные могут передаваться только тем сервисам, которые необходимы для работы сайта, обработки платежей, хранения информации, аналитики и обеспечения безопасности.

10. Безопасность

Мы принимаем разумные технические меры для защиты данных пользователей, включая хеширование паролей, использование защищённых соединений и ограничение доступа к серверным ключам.

Однако ни один способ передачи данных через интернет или хранения информации не может быть гарантированно защищён на 100%.

11. Права пользователя

Пользователь может запросить:

  • доступ к своим данным;
  • исправление неверной информации;
  • удаление аккаунта;
  • удаление персональных данных, если это возможно и не противоречит закону;
  • отзыв согласия на обработку данных.

Для таких запросов пользователь может связаться с администрацией сервиса по контактному email.

12. Данные резюме

Пользователь самостоятельно отвечает за информацию, которую вводит в резюме. Не рекомендуется указывать в резюме чувствительные данные, если это не требуется для цели его создания.

Введённые данные используются для отображения резюме в редакторе и создания PDF-файла.

13. Данные несовершеннолетних

Сервис не предназначен для сбора данных детей младше 13 лет. Если вы считаете, что такие данные были переданы нам случайно, свяжитесь с нами для их удаления.

14. Изменения политики

Мы можем периодически обновлять настоящую Политику конфиденциальности. Актуальная версия всегда публикуется на этой странице.

Продолжение использования сайта после изменения Политики означает согласие пользователя с обновлёнными условиями.
